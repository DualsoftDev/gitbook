# DS 다운로드

{% hint style="success" %}
DS를 이용해 주셔서 감사합니다. 필요한 자료를 아래에서 다운로드하실 수 있습니다.
{% endhint %}

{% embed url="http://dualsoft.co.kr/download/" %}

{% hint style="info" %}
**다운로드 방법**

* 상단 \[파일목록] 을 클릭 후 다운로드 페이지로 이동합니다.
*   \[Dualsoft Software] 탭에서 Setup DS 파워포인트 Addin의 다운로드 버튼을 누릅니다.

    <figure><img src="../../.gitbook/assets/제목 없음1111.png" alt=""><figcaption></figcaption></figure>
* 다운로드한 파일을 확인하고 압축을 푸세요.
*   압축을 풀고 exe 파일을 우클릭 하여 관리자권한으로 실행하세요.

    <figure><img src="../../.gitbook/assets/제목 없음2222.png" alt=""><figcaption></figcaption></figure>
* 다음을 눌러 설치를 진행하세요.
*   만약 윈도우 PC보호 화면이 뜬다면 밑줄 친 추가 정보를 클릭 후 실행 버튼을 눌러서 진행합니다.

    <div><figure><img src="../../.gitbook/assets/화면 캡처 2025-11-10 103247.png" alt=""><figcaption></figcaption></figure> <figure><img src="../../.gitbook/assets/화면 캡처 2025-11-10 103311.png" alt=""><figcaption></figcaption></figure></div>
{% endhint %}

{% hint style="info" %}
*   PowerPoint를 실행하여 추가기능(DUALSOFT)탭이 정상적으로 설치되었는지 확인하세요.

    <figure><img src="../../.gitbook/assets/슬라이드4 (1).PNG" alt=""><figcaption></figcaption></figure>
*   설치 또는 사용 중 궁금한 점이 있다면 추가기능탭 > DS About 영역의 Version을 클릭 후 "기본 메뉴얼을 여시겠습니까?" 메세지 박스가 뜨면 확인을 눌러 매뉴얼PPT를 불러옵니다.

    <figure><img src="../../.gitbook/assets/image (31) (1).png" alt=""><figcaption></figcaption></figure>
{% endhint %}

{% hint style="danger" %}
**유의사항**

* 다운로드한 파일은 무단 복제 및 배포가 금지됩니다.
* 최신 버전을 확인하고 사용해 주세요.
* 다운로드 관련 문제가 발생하면 고객센터로 문의해 주세요.
{% endhint %}
