# CCTV 영상

### ✅ 기능 설명

**CCTV 영상 기능은 실제 설비 동작 현장을 실시간으로 확인**하고,\
센서/기기 상태, 제어 신호, 포지션 정보 등을 **영상 위에 텍스트 및 그래픽으로 오버레이**하여\
공정 흐름과 장비 상태를 시각적으로 통합 파악할 수 있도록 지원하는 기능입니다.

### 📌 실무 TIP

* 설치된 카메라 위치는 공정의 핵심 포인트(Robot, Clamp, Sensor 등)를 기준으로 배치
* 영상 위 오버레이되는 텍스트와 Tag는 설정 메뉴에서 **수정 및 선택 가능**
* 이 기능은 **이력 분석 및 이상 보기** 모듈과 함께 사용할 때 효과 극대화
* 복수 카메라 동시 출력 가능 (화면 분할 또는 선택)
